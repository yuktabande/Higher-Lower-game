# Higher-Lower Game 🎮
A simple command-line game where players guess which celebrity has a higher follower count on social media.

## 🛠 Features
- Compare the follower counts of two random celebrities.
- Keep playing until you make an incorrect guess.
- Fun and interactive CLI experience!

## 📂 Project Structure
- main.py → Core game logic.
- game_data.py → Contains the dataset of celebrities and their follower counts.
- art.py → ASCII art for game visuals.

## 🚀 How to Run
- Clone the repository:
```bash
git clone https://github.com/yuktabande/Higher-Lower-game.git
cd Higher-Lower-game
```
- Run the game:
```bash
python main.py
```
## 📝 Requirements
```
Python 3.x
```

## 🎯 How to Play
- The game presents two celebrities.
- You guess which one has more followers.
- If correct, you continue. If wrong, the game ends.
Enjoy the game! 🚀
